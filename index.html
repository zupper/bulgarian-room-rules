This is the official set of rules for the Bulgarian Developers StackOverflow chatroom. Read them and abide by them, or risk the wrath of the room owners!

What are the rules?

Basically stripped the Android room rules, but here it goes:

  • Speak either English, or Bulgarian.
  • Do not paste code into chat, instead use a site such as Ubuntu Pastebin or a GitHub Gist.
  • When you Ping (preface a username with an "@"), do it reasonably.
  • If you have an issue which you cannot solve after research, write your question on Stack Overflow and paste a link to it in the channel, it will be automatically formatted to allow people to read it.
  • If no one helps, we're either busy or we don't know.

How to get write access?

The room is generally read-only and write access has to be granted before one can write in the room. To get access, you need to cover all of the following:

  • Reputation > 100
  • Custom user name (and NOT user12345).
  • You have asked and answered reasonable questions on Stack Overflow.
